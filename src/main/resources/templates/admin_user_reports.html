Instructions:

Use this form to generate reports for an individual user's contributions to the sonnet database. Each report includes the author, title, and date each sonnet was added to the database.

You must enable pop-ups for the reports to work.

To use the form:

  • Select the desired user's username in the first box.
  • Enter the start of your desired date range into the "Start" box.
  • Enter the end of your desired date range into the "Finish" box.
  • Click "Get Report" and a new window will pop-up. The window always appears to be loading, but as soon as the window pops up the report is finished. Ignore the apparent loading.
