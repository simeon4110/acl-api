Moment's Monuments: The Sonnet Database

This database, named for Dante Gabriel Rossetti's The House of Life, offers every sonnet in the English language, from the sixteenth century to the present, in interoperable formats. The database is open to public use. However, due to stringent data integrity requirements, only authorized users may contribute to the database.

The ACL sonnet database provides a public RESTful API interface to facilitate programmatic database queries for academic work in the digital humanities. For more information, or to request researcher / contributor access, please email Michael Ullyot at ullyot@ucalgary.ca with a brief description of what you intend to use the database for and why you need non-public access.

Text Encoding Initiative

The Text Encoding Initiative (TEI) is a consortium which collectively develops and maintains a standard for the representation of texts in digital form. Its chief deliverable is a set of Guidelines which specify encoding methods for machine-readable texts, chiefly in the humanities, social sciences and linguistics.

Python Database Connector

We provide and maintain a Python class for easilly connecting to our database via the RESTful API. This class provides basic query functions and JSON object serialization. This allows a user to skip most of the data parsing and jump straight into NLTK or NLP analysis of the data.
